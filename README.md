# Credit-Card-Dashboard
## Project Overview

This project focuses on building a Credit Card Financial Dashboard to analyze customer behavior, transaction patterns, and revenue insights using SQL and Power BI.

The dashboard provides real-time, data-driven insights to help stakeholders monitor performance and make better business decisions.

## Objective

To develop an interactive dashboard that:

Tracks key financial metrics
Analyzes customer spending behavior
Monitors weekly and yearly trends
Provides actionable business insights

## Tech Stack
SQL (PostgreSQL) – Data storage & querying
Power BI – Data visualization & dashboard
CSV Files – Data source

## Dataset Information

This project uses four CSV files containing transaction and customer data:

credit_card.csv – Main credit card transaction data
cc_add.csv – Additional weekly transaction data
customer.csv – Customer demographic details
cust_add.csv – Additional customer data

## Key Insights
Total Revenue: 55.4M+ | Transactions: 45M | Interest: 7.9M
Revenue grew 28.8% WoW with stable quarterly performance
Male customers contribute slightly higher revenue than females
Blue & Silver cards account for ~93% of total transactions
Top states (TX, NY, CA) contribute ~68% of revenue
Highest spending in Bills, Entertainment, Fuel
Core customers: Age 30–50 & High-income group
Activation Rate: 57.5% | Delinquency Rate: 6.06%
